<CourseBlock id={'whats-inside-research-thinking'} divider={{ color: 'var(--yellow)', topImage: '/nlp-course/ico/bulb_empty.webp' }}>
  <CLeft>
    ## Research Thinking
    Learn to think as a research scientist:
    - find flaws in an approach,
    - think why/when something can help,
    - come up with ways to improve,
    - learn about previous attempts.

    It's well-known that you will learn something easier
    if you are not just given the answer right away, but if you think about it first. Even
    if you don't want to be a researcher, this is still a good way to learn things!
  </CLeft>
